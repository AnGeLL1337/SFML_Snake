# SFML_Snake
Simple snake game writed in C++ using SFML library.

# Installation
1. Download project and connect with SFML library. You will need these modules:
  - sfml-system
  - sfml-graphics
  - sfml-window
2. Compile project and enjoy.

# Ideas for improvement
